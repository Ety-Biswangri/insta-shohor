Some Important Concepts of JavaScript

JavaScript is a high level, single threaded, interpreted programming language. For execution of JavaScript codes, every browser provides a JavaScript engine. For example, Google Chrome provides V8 engine for executing JavaScript codes.

At first, V8 engine takes the JavaScript source code, then parses it and turns that into an Abstract Syntax Tree and then generates Bytecode from that tree. V8 engine uses TurboFan which follows Just In Time compilation and generates machine code from the Bytecode. Finally, This machine code runs on the CPU. So, this is how JavaScript works.

Local Storage and Session Storage are two mechanisms of Web Storage API. The differences of this two mechanisms are:

  • Local storage stores data in browser without any time limitation. On the other hand, Session storage stores data in browser for a limited duration.
  • Local storage stores data until the data is explicitly removed by the users. On the contrary, Session storage stores data until the tab or the browser is closed.
  • In local storage, data is accessible from all tabs and window with same origin. Whereas, in session storage, data is only accessible from the current tab of a browser as long as it is open.
